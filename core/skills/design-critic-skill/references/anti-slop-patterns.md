# Anti-Slop Patterns

## Common Weak Signals

- Generic gradient background carrying the whole design.
- Random decorative orbs or blobs.
- Nested cards without structural need.
- Huge hero type inside operational tools.
- Unstable button/card dimensions.
- Text overlapping icons, counters, or neighboring blocks.
- One-note palette without semantic roles.
- Stock-like imagery that hides the real product, place, or object.

## Review Rule

Name the observable issue and the user-facing consequence. Then provide a concrete fix.
